# 🌐 Subnet Calculator

A simple subnet calculator built with **C++** and the **Qt Framework**.

---

## 🚧 Project Status

**This project is currently in active development.**

The application is still a work in progress and is being developed as both a networking utility and a way to expand knowledge of C++ and Qt application development.

Features, functionality, and the user interface may change frequently as development continues.

---

## 📖 About

Subnet Calculator is a desktop application designed to simplify common IPv4 subnetting tasks through an easy-to-use graphical interface.

The goal is to provide a lightweight tool that helps users quickly calculate subnet information while also serving as a practical software development project.

---

## ✨ Current Features

- Qt-based graphical user interface
- IPv4 address input
- Subnet mask calculations
- CIDR notation support
- Binary subnet mask display

---
